# Changelog — Story Magic

## [1.0] — Initial release

- Character, setting, and theme selection
- Multi-page story generation via Claude AI
- Per-page illustration generation via DALL-E 3
- Magic book UI with CSS animations (twinkle stars, floating book, gold shimmer)
- PDF export in landscape A4 storybook layout (ReportLab)
- Story history tracker saved to Excel
- Stories saved as JSON files in `stories/` folder
- Deep space magic dark theme (Fredoka One, Nunito, Cinzel fonts)
