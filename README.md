# Story Magic

AI-powered storybook creator for kids aged 6–12 by VisualizePro. Choose a character, setting, and theme — get a fully illustrated story generated by Claude AI and DALL-E 3, downloadable as a PDF storybook.

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | Streamlit |
| Story AI | Anthropic Claude API (claude-sonnet-4-5-20250929) |
| Image AI | OpenAI DALL-E 3 |
| PDF export | ReportLab |
| Tracker | openpyxl |

## Setup

1. Install dependencies
   ```
   pip install -r requirements.txt
   ```
2. Ensure `.env` exists at `C:\Users\madel\.env` with:
   ```
   ANTHROPIC_API_KEY=your_key_here
   OPENAI_API_KEY=your_key_here
   ```
3. Run the app
   ```
   streamlit run storybook_app.py
   ```
4. Open http://localhost:8501

## Project Structure

```
storybook_app/
├── storybook_app.py        Main Streamlit app
├── requirements.txt        Python dependencies
├── stories/                Saved story JSON files
├── storybook_tracker.xlsx  Story history tracker
```

## Features

- Character, setting, and theme selection
- AI-generated story pages via Claude
- AI-generated illustrations via DALL-E 3 (one per page)
- Magic book UI with animations (twinkle, float, shimmer)
- PDF export (landscape A4 storybook layout)
- Story tracker saved to Excel
- Target age range: 6–12 years
